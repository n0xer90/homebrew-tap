# Homebrew Tap

Custom Homebrew tap for macOS utilities by [n0xer90](https://github.com/n0xer90).

## Usage

```bash
brew tap n0xer90/tap
```

## Available Apps

| App | Install | Description |
|-----|---------|-------------|
| [MemoryBar](https://github.com/n0xer90/MemoryBar) | `brew install --cask memorybar` | Lightweight menu bar memory & CPU monitor |
